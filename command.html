Command Line!

  • pwd - Print Working Directory
  • hostname - my computer's network name
  • mkdir - make directory
  • cd change directory
  • rmdir - remove directory
  • pushd - push directory
  • popd - pop directory
  • cp - copy file or directory
  • mv - move file or directory
  • less - look through all contents of a file
  • echo - write to a file
  • cat- contents of the file
  • ls - list directory
  • ls -a - all contents including hidden files
  • ls -l - all contents long format
  • ls -t - orders files & directories by time they were last modified
  • ls -alt - multiple options can be used at once
